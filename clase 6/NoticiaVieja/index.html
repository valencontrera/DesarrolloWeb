Impresionante: este es el truco para emojis "enormes" de WhatsApp

Una nueva forma de enviar emoticones para que sean más notarios dentro de la aplicación de mensajería instantánea de Meta. El paso a paso para sorprender a tus contactos

Cómo es el truco para emojis "enormes" de WhatsApp.

WhatsApp tiene todo tipo de animaciones como sticker y emojis para adornar los mensajes y cada vez desarrolla nuevas funciones para crearlos, hacerlos personalizados o en este caso más grandes.

Entre los trucos que tiene la aplicación de mensajería instantánea de Meta y que pocos usuarios conocen está disponible el de enviar emojis “gigantes” dentro de los chats. La función aplica tanto para celulares con Sistema Operativo de Android como de iOS.

Los emojis revolucionaron las conversaciones haciéndolas más atractivas, interactivas y divertidas. En la actualidad los mismos tienen un tamaño bastante pequeño por lo cual este truco es genial para utilizarlo.

Cómo enviar emojis "enormes" de WhatsApp

Para realizar este cambio de tamaño en los emojis es necesario descargar la aplicación Gboard, que es un teclado desarrollado por Google, para dispositivos móviles.

  • Una vez instalada, se debe elegir como el teclado predeterminado del dispositivo.
  • Luego ingresar a un chat de la aplicación de mensajería de Meta.
  • Se debe activar el teclado como si se fuera a escribir un mensaje.
  • Seleccionar el ícono de emojis Gboard. En la parte de los emojis animados,oprimir la opción para ver los emojis tradicionales.
  • De esta forma aparecerán los mismos emojis de WhatsApp, con la posibilidad de mezclar dos de ellos. Y de un tamaño mucho más grande para poder enviar en todas las conversaciones.
